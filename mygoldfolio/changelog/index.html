v2.0.4 APK Apr 2026

Better error messages, proactive permission prompts, one-time schedule support, and merged settings sections.

  • ✓ Friendlier alarm permission prompt — Amber "One more step" banner inside Background Fetch card + proactive popup when enabling. Mentions no battery impact.

v2.0.3 APK Apr 2026

Reliability fixes from S24 Ultra + S9+ device testing. Alarms, exports, and bottom sheets now work correctly across all screen sizes and Android versions.

  • ✓ Alarm permission guidance — Red warning banner + "Grant Permission" button when exact alarms are denied on Android 12+. Auto-reschedules when permission is granted.
  • ✓ Test alarm button — Schedule a 2-minute test alarm to verify exact alarm delivery and method used.
  • ✓ Export fallback — "Save" gracefully falls back to share sheet on Samsung/Android 10 devices where SAF folder access fails.
  • ✓ Schedule modal fix — "Add" button and time preview no longer clipped on smaller screens (S9+).
  • ✓ Price update sheet fix — "Dismiss" button no longer hidden behind Android navigation bar.
  • ✓ Enhanced diagnostics — Exact alarm permission status in App Info, boot receiver logging, BG debug logs enabled.
