App Description

MyGoldFolio is a smart gold tracker for Android. Set it once, forget it — the app watches prices in the background and alerts you when it matters. Features include live prices from 3 sources with automatic fallback, per-holding P&L tracking, tax-free countdown, trend-aware what-if scenarios, and background price alerts.

All data is stored exclusively on the user's device — there are no accounts, no cloud sync, and no data collection of any kind. The app is free to download with no ads and no in-app purchases.

MyGoldFolio is built with React Native (Expo SDK 54, New Architecture) and is available for Android 10 and above.

Platform: Android 10+ · Price: Free · Version: 2.0.1 · Size: ~12 MB · Category: Finance
