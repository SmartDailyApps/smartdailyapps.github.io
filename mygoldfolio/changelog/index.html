Changelog

Release notes for every public version of MyGoldFolio

v1.8.2 April 2026
  • Per-holding sell redesign — each sell is now attached to a specific holding rather than drawn from a shared pool
  • Partial-sell warning shown for physical item types (bar, coin, jewellery)
  • Removed FIFO/LIFO pool allocation toggle from Settings (replaced by per-holding sell)
  • Removed pool sell button from Portfolio header
  • Sell history and realised P&L now clearly tied to individual holdings
v1.8.1 March 2026
  • Persist section collapse state — expanded/collapsed state of all sections is now remembered across app restarts
  • Configurable defaults for collapsible sections via developer constants
  • Store migration v13 → v14
v1.8.0 March 2026
  • Collapsible sections on Dashboard: Portfolio Overview, Sold Overview, Goals, Key Metrics, Tax Status, Price Trend
  • Collapsible Holdings and Sold sections in Portfolio (list and compact views)
  • Show-more for holdings lists with more than 5 entries
  • Collapsible hero detail section in Prices tab
  • Top-5 scenarios shown by default in Scenarios tab with expand button
  • Removed redundant sold proceeds banner from Portfolio
